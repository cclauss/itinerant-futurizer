See: http://python-future.org/futurize_cheatsheet.html

* git clone [url]
* cd [reponame]
* git checkout -b futurize-stage-1
* pip install future
* futurize --stage1 -w **/*.py
* git commit --all -m "futurize --stage1 -w **/*.py"
* git push --set-upstream origin futurize-stage-1
